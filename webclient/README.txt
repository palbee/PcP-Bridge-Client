Requirements:

dojo-release-1.6.1 must be installed. Download the Dojo Toolkit Release from http://dojotoolkit.org/download/ and install it where the web server can find it.


To install the web client.
1) Put blank.html and pcp_webclient.html  where web pages are served from. 
2) Put pcp.js in the javascript directory. 
3) In pcp_webclient.html edit the line 'dojoBlankHtmlUrl : "blank.html",' to reflect the absolute url for 'blank.html'. 
4) In pcp_webclient.html edit the script for the dojo base location. 
5) in pcp_webclient.html edit the line '<script src="pcp.js" type="text/javascript"></script>' to reflect the url for 'pcp.js'.
